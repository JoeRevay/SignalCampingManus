==================================================

Create a new file:
client/src/lib/affiliate.ts

1. import affiliateProducts.json
2. create types/interfaces for affiliate products
3. create helper functions:

A) getActiveAffiliateProducts(slot: string)
- returns active products for a slot
- sorted by priority ascending

B) getPrimaryAffiliateProduct(slot: string)
- returns the first active product for the slot
- returns null if none

C) getAffiliateSlotsForCampground(campground)
This is the key logic.

Use existing campground fields to decide which slots to show.

Use these rules:

- always include "portable_power"

- if remote_work_score is defined and >= 85, include "starlink_accessory"
- do not return duplicates
- maximum 3 slots returned

Make this logic safe for undefined/null values
